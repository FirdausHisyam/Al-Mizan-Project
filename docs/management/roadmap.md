# Product Roadmap

## Phase 1: Foundation & Planning (Current)
-   **Goal**: Establish a solid, secure architectural base.
-   **Deliverables**:
    -   Project Charter & SAD.
    -   Tech Stack Finalization (Rust/SurrealDB).
    -   Dev Environment Setup.

## Phase 2: Core Backend Implementation
-   **Goal**: Secure API for accessing Islamic texts and relationships.
-   **Deliverables**:
    -   Rust API with Authentication.
    -   SurrealDB Schema Finalization.
    -   Data Ingestion Scripts (Quran/Hadith).

## Phase 3: Visualiser Development
-   **Goal**: Interactive frontend for exploring abrogations.
-   **Deliverables**:
    -   SolidJS Application.
    -   Graph Visualization (Cytoscape/D3).
    -   Search & Filter Functionality.

## Phase 4: Verification & Hardening
-   **Goal**: Ensure correctness and security.
-   **Deliverables**:
    -   Penetration Testing.
    -   Scholar Verification of Data.
    -   Performance Optimization.
